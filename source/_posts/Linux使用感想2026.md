主力机换成 Linux 用了有一阵子了，谈谈我的感受。

# 梦开始的地方
两年前更新了 Win11，发现远程桌面有 bug，被控的 Win11 电脑会不定期死机。于是降级 Win10，第二天起来发现自动更新 Win11 了。上半辈子没说的脏话在这一刻全喷出来了。

当然以前没少见过说"自动更新有办法关的，你菜就别怪别人"。但是这合理吗？首先，作为这台电脑的所有者，小版本不说了，大版本更新不用取得我同意吗？其次，Win 不是喜欢说自己用户界面易于操作吗，为什么关自动更新要去调注册表？咋不说该通过逆向让 Win 变成想要的模样呢？

啊反正就是这样，hsn 讨厌 MS 对用户的态度。

# 蜜月期

第一次尝试是在移动硬盘装 Ubuntu。一切顺利，没啥问题，用了一段时间，PyCharm 也能正常使用。于是货比三家换 Manjaro 了，也是一切顺利。但那时比较闲，蓝条也比较长，找了个时间折腾把电脑又换成 Arch 了，当时用的 KDE Plasma 5。一切顺利，花了半小时进 sddm 了，因为之前的 /home 完全拷过来了，所以没啥迁移的负担，重新把软件装好又继续用了。

不得不说当年也当过嘉豪，在班级大屏上尝试装了 Ubuntu，但是因为当时还没有这么多开源的白板，所以放弃了这个想法。

# 磨合
后来折腾过输入法，给台式机装的时候折腾过 NV 驱动，折腾过 Hyprland，也尝试换成 Fedora，滚炸过，迁移过。无数次觉得这系统咋这么折腾啊，但是依然坚持了下来。

# 养老
折腾着折腾着发现，稳定好用才是真，自己已经不像当年这么多蓝条了。输入法干脆就 iBus+Libpinyin。桌面也换成 Gnome，唯独用了 Hyprland 后离不开平铺管理器了，无奈加上了 PaperWM。系统也固定下来用 Arch，最熟悉，最好修，也足够新。不再折腾啥 Flatpak，因为历史遗留问题，root 特别小。啊总之，hsn 逐渐在水塘里仰泳了起来。

# 感想
终于回归正题了吗？这几年用 Linux 学到很多，也变成了典型的 Linux 思维。Linux 对于 hsn 来说不是平替，而是很不错的生产力工具，hsn 所有需要的他都有，甚至比 Win 更多。Linux，非常自由，可以很大程度上决定自己电脑上有什么，没有什么，比如 Edge 不想用了可以卸。Linux 对很大一部分开发者来说体验是一流的，啊反正经常有遇到 Linux 和 Mac 上能跑起来的东西在 Win 上就是有问题，印象深刻是 Python 的 async subprocess 在 Win 上不能用，好像是要切换老版本的 Loop 才能跑。可能未来很长一段时间里，hsn 还是会抱着 Linux 不放。

# 反向感想
Win 真对得起 hsn 喊他一句"垃圾系统"，他有优点，但是，突出点越来越少了。Linux 不再是程序员/服务器专属，甚至由于 V 社的大手，也逐渐可以打游戏了。今天和群友聊天聊到了 Win，问 AI：Win 有什么优点。紧接着 hsn 把所有 "优点" 反驳了一遍。Mac 和 Linux 不一定适合所有人，但是 Win，确实该改进了。

# 小趣事
Fedora 没再用是因为，刚安装好，提示可以更新，点击更新，炸了。就很莫名其妙，于是跑回去用 Arch 了，至少炸了还会修。

平铺桌面管理器，让除了 hsn 之外所有人都不能轻松用这台电脑。
